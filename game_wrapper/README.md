# Auto-Scientist Game Wrapper: DEEP SEARCH
# "The Casual Odyssey Update"

A gamified and immersive interface for exploring the bioplausible hyperparameter space.

## Controls
- **WASD**: Fly
- **Mouse**: Look (ESC to toggle mouse)
- **L-SHIFT**: **WARP BOOST**
- **SPACE**: **SCAN SECTOR**
- **F**: **LAUNCH PROBE**
- **V**: **CYCLE VISORS**
- **P**: **PHOTO MODE** (Toggle UI)
- **TAB**: Cycle Vehicle
- **T**: Cycle Mission Task
- **R**: Reset Position

## Casual Features
- **Gravity Assist**: High-performance experiments exert a gentle gravitational pull, guiding you toward scientific discovery.
- **Sector Naming**: Explore procedurally named regions like "The Void Limit" or "Alpha Cluster (High Capacity)".
- **Trails**: Paint your path through the stars.
- **Photo Mode**: Remove the HUD to take beautiful screenshots of your discoveries.

## Sandbox Features
- **Autonomous Probes**: Launch unlimited probes to create a network of autonomous explorers.
- **Atmospheric Nebula**: The space is populated with procedural nutrient clouds.
- **Visor Modes**: Standard, Performance (Heatmap), Chrono.

## Core Features
- **Mission Log**: Casual objectives to guide your journey.
- **Generative Audio**: Procedural ambient music.
- **Inspector**: Point at any star for scientific data.
- **Physics**: Newtonian flight model.

## Running
```bash
./play_scientist.sh
```

# Auto-Scientist Game Wrapper: DEEP SEARCH
# "The Stellar Garden Update"

A relaxing, open-ended sandbox to cultivate scientific discovery.

## Controls
- **WASD**: Fly
- **Mouse**: Look (ESC to toggle mouse)
- **Left Click**: **LINK STARS** (Select/Connect)
- **Right Click**: **CLEAR LINKS**
- **L-SHIFT**: **WARP BOOST**
- **SPACE**: **SCAN SECTOR** (Plant Experiment)
- **F**: **LAUNCH PROBE**
- **V**: **CYCLE VISORS**
- **P**: **PHOTO MODE**
- **+/-**: **TIME WARP** (Change Sim Speed)

## Garden Features
- **Constellations**: Click on stars to draw lines between them. Create your own shapes, map out clusters, or just connect the dots.
- **Musical Physics**: Fly close to stars to hear them sing. The pitch depends on their hyperparameters.
- **Living Atmosphere**: The nebula colors shift from Angry Red to Serene Blue as you discover higher-accuracy models.
- **Time Control**: Speed up the simulation to watch your garden grow faster, or slow it down to drift.

## Casual Features
- **Gravity Assist**: Gentle pull towards high-grade discoveries.
- **Trails**: Leave a light path behind you.
- **Sector Naming**: Procedural region names.
- **Objectives**: Optional creative and scientific goals.

## Core Features
- **Mission Log**: Casual objectives to guide your journey.
- **Generative Audio**: Procedural ambient music.
- **Physics**: Newtonian flight model.

## Running
```bash
./play_scientist.sh
```
